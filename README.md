# TicTacToe-Java-AWT 🎮

A simple desktop-based Tic Tac Toe game built using Java AWT.  
This is a 2-player game where players take turns marking X and O on a 3×3 grid.

---

## 📌 About the Project
This project demonstrates how to build a basic GUI application in Java using AWT (Abstract Window Toolkit).  
It includes game logic, win detection, and a simple interactive interface.

---

## 🚀 Features
- 3x3 interactive game board
- Two-player gameplay (X and O)
- Automatic winner detection
- Game stops when a player wins
- Simple and lightweight GUI using AWT

---

## 🛠️ Technologies Used
- Java
- AWT (Abstract Window Toolkit)
- Event Handling (ActionListener)
